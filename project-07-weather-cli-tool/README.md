# Project 07: Weather CLI Tool

---

## Overview

A simple CLI tool to fetch weather information using an API.

---

## Features

- Get current weather
- Supports automatic location detection
- Fetch data using curl

---

## Usage

```
./weather.sh
```

---

## Tech Used

- Bash
- curl
- wttr.in API
