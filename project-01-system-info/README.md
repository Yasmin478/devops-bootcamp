# System Information Bash Script

A simple Bash script that displays important system information in a clean and readable format.

This project is part of my DevOps learning journey.

---

## Features

The script displays:

- Hostname
- Operating System
- Kernel Version
- CPU Architecture
- Total Memory
- Available Memory
- Memory Usage
- Disk Usage (Root partition)
- System Uptime
- Current Logged-in Users
- Current Date & Time
- IP Address

---

## Technologies Used

- Bash
- Linux CLI tools (`hostname`, `uname`, `free`, `df`, `uptime`, `who`, `date`)
- `awk` for data parsing
